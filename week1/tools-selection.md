# Tools and Technology Selection

## Target Applications
- DVWA (Damn Vulnerable Web Application)
- Juice Shop / bWAPP (for extended testing)

## Security Testing Stack
- Python (core implementation language)
- Requests / Selenium (interaction and scanning)
- BeautifulSoup (HTML parsing)

## AI and Automation Stack
- Python-based rule engine for intelligent decision-making
- pandas and numpy for response data analysis
- scikit-learn (future phase) for anomaly detection and classification

## Environment
- XAMPP (DVWA setup) or Docker (Juice Shop)
- Localhost-based controlled testing environment

## Reason for Selection
- Compatible with automated security testing
- Supports AI-driven analysis and extensibility
